# Merry Christmas
> 네이버 블로그의 From.블로그씨로부터 온 '트리 만들기' 주제의 메세지를 보고 동기를 얻어 만들게 된 위젯입니다.

## 기능
- 이미지 설정(배경 없는 이미지 추천)
- 메세지 설정

이미지와 메세지를 설정 후, 위젯을 띄우면 화면에 위젯이 표시됩니다.
항상 화면의 최상단에 표시되기 때문에 다른 작업을 하더라도 가리지 않습니다.
설정한 이미지와 그 아래로 검은 배경에 형광 녹색의 글씨체로 설정한 메세지가 표시됩니다.
(꼭 트리가 아니더라도 원하는 이미지를 항상 PC 화면에 띄워놓을 수 있습니다.)
